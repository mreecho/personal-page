About Me

Aspiring data professional passionate about turning raw data into actionable insights. Currently pursuing my degree at Botswana Accountancy College while developing web solutions and creating technical content.

With hands-on experience in Python and SQL, I specialize in transforming complex datasets into clear visualizations using tools like Power BI and Tableau. My journey in tech began with building responsive websites, which evolved into a passion for data storytelling.

I've led several academic projects including a predictive model for local agricultural yields and a dashboard tracking Botswana's economic indicators. When not coding, I produce tutorial videos breaking down data concepts for beginners.

Committed to lifelong learning, I'm currently exploring machine learning applications in financial analytics. My goal is to bridge the gap between technical data analysis and business decision-making in African markets.
